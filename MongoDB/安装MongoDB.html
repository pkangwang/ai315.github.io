下载MongoDB的.msi安装包。安装时需要指定数据目录，也可以使用默认的目录。

启动MongoDB ​

在命令中运行mongod启动：

$   mongod

如果是Windows系统，执行：

$   mongod.exe

数据目录

默认数据目录为/data/db，Windows系统中为当前卷的\data\db。如果数据目录不存在，将无法启动，因此启动之前要检查这个目录是否存在，不存在要新建。

默认监听27017端口。如果端口被占用，将无法启动，常见情况是另一个MongoDB实例正在运行。

在命令中按Ctrl + C停止MongoDB实例。

要连接其它机器或端口的mongod，需要指定主机名、端口和数据库：

mongo some-host:30000/otherDB

MongoDB shell ​

在mongod服务启动后，就可以在另一个命令行启动shell：

$ mongo

shell就是MongoDB的客户端。shell会自动尝试连接本地的MongoDB服务端。因此请确保mongod已启动。

shell可以运行任意JavaScript程序。

要查看当前的数据库，输入：

shell
>  db
test

要连接一个数据库，输入：

shell
>  use  blog
>  db
blog

要访问集合，输入：

db.blog.users

这会返回users集合。
